# Configurations Directory

This directory contains various configuration files used in the project.
The configurations are organized into subdirectories based on their type and purpose.

## Directory Structure
- `amplifiers/`: Contains amplifier configuration files.
- `fibers/`: Contains fiber configuration files.
- `gnpy_parameters/`: Contains GNPy parameters files.
- `networks/`: Contains network related files, including world map, ROADM icon and topologies.
- `spectrum`: Contains spectrum configuration files.
- `transceivers/`: Contains transceivers configuration files.
- `simulation_parameters/`: Contains simulation parameter files.
  - `GA_parameters/`: Genetic Algorithm parameter files.
  - `power_optimization_parameters/`: Power optimization parameter files.
